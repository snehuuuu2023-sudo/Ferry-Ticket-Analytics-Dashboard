# Ferry Ticket Analytics Dashboard

An interactive analytics dashboard developed using Streamlit for ferry passenger analysis, ticket sales monitoring, redemption tracking, machine learning prediction, and real-time simulation.

## Features
- Interactive Filters
- KPI Dashboard
- Passenger Trend Analysis
- Real-Time Simulation
- Machine Learning Prediction
- Download Reports
- Interactive Visualizations

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

## Run Locally

```bash
streamlit run app.py
